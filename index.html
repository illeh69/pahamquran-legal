Kebijakan Privasi

Terakhir diperbarui: 27 Juni 2026

PahamQuran tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna dalam bentuk apapun. Aplikasi ini berjalan sepenuhnya secara offline di perangkat Anda.

1. Informasi yang Kami Kumpulkan

PahamQuran tidak mengumpulkan informasi pribadi apapun dari pengguna. Tidak ada akun, tidak ada pendaftaran, tidak ada login yang diperlukan untuk menggunakan aplikasi ini.

2. Penggunaan Data

Semua data aplikasi — termasuk kosakata Al-Qur'an, bentuk tashrif, contoh ayat, dan data Asmaul Husna — tersimpan secara lokal di dalam aplikasi dan tidak dikirimkan ke server manapun.

3. Konektivitas Internet

PahamQuran tidak memerlukan koneksi internet untuk berfungsi. Aplikasi ini dirancang untuk digunakan sepenuhnya secara offline. Tidak ada data yang dikirim keluar dari perangkat Anda.

4. Layanan Pihak Ketiga

Aplikasi ini menggunakan layanan Text-to-Speech bawaan perangkat (sistem operasi iOS/Android) untuk fitur pelafalan Arab. Layanan ini dikelola oleh Apple/Google sesuai kebijakan privasi masing-masing platform. PahamQuran tidak mengontrol atau mengakses data dari layanan tersebut.

5. Data Anak-Anak

PahamQuran aman digunakan oleh semua kalangan usia, termasuk anak-anak. Kami tidak mengumpulkan informasi dari siapapun, termasuk anak-anak di bawah usia 13 tahun.

6. Perubahan Kebijakan Privasi

Jika ada perubahan pada kebijakan privasi ini, kami akan memperbarui tanggal "Terakhir diperbarui" di halaman ini. Kami mendorong pengguna untuk meninjau halaman ini secara berkala.
